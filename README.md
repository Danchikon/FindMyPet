# FindMyPet | [link]()

## Instalation
1. clone this repo
```git
  git clone https://github.com/Danchikon/FindMyPet
```
2. Open terminal in ./src/ directory
```cmd
  cd src/
```

3. Start node.js
```node
  npm start
```

4. Open your browser and navigate to:
```cmd
  http://localhost:3000
```

5. Ready to use.
